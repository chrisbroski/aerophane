Google's material design is a visual style made for mobile user interfaces. Aerophane is a template that embraces this style to create an ultra light starting point for hybrid mobile web development.

Aerophane strives to have a small footprint, run blazingly fast, and be exceedingly easy to modify while also implementing the most common features used in modern mobile UIs.
