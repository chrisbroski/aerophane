Determinate Progress Indicator

Adding the progress-determinate class to an element will add a settable progress bar to the bottom. The length of the bar is set with the CSS variable "width". For example: 

<header class="progress-determinate" style="--width:50%">
</header>

JavaScript on the page is setting the width dynamically.

Indeterminate Progress Indicator

Adding the progress-indeterminate class to an element will add a pure CSS animated progress bar to the bottom.

Spinner

The circular indeterminate spinner above is an SVG element with the spinner class. Just paste in this markup to create your own. 

<svg class="spinner" width="32px" height="32px" viewBox="0 0 66 66" xmlns="http://www.w3.org/2000/svg">
    <circle fill="none" cx="33" cy="33" r="30"></circle>
</svg>

Like the linear indeterminate progress indicator, this is a pure CSS animation. No JavaScript is needed.
